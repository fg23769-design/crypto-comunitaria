Contribuir
----------

Gracias por querer contribuir. Por favor sigue estos pasos:

1. Lee el README.
2. Abre una Discussion si tienes una idea general.
3. Para cambios de código: fork -> branch -> PR. Usa el formato de commit claro.
4. Crea issues claras (descripción, pasos para reproducir, propuesta de solución).
5. Reglas básicas:
   - Sé respetuoso.
   - Transparencia sobre intereses económicos.
   - No aceptaremos trabajo que no pase auditoría de seguridad si afecta la moneda o la custodia.
